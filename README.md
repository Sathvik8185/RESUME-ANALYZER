# RESUME-ANALYZER

A Resume Analyzer is a web-based application that evaluates candidate resumes by comparing them with a given job description. The system extracts key information such as skills, experience, education, and relevant keywords from the resume and measures how well they match the job requirements. The analyzer provides a match score.

# Frontend JavaScript Task

## Features

- Form validation using JavaScript
- PDF file type checking
- Dummy backend response simulation
- No backend integration

## How to Run

1. Clone the repo
2. Open index.html in a browser

## Notes

- All logic is handled using vanilla JavaScript
- Dummy data is used to simulate backend responses
